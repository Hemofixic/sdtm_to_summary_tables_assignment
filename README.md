В данном репозитории находится код для преобразования SDTM‑файлов в несколько таблиц, содержащих результаты клинических исследований.
Исходные SDTM‑данные не включены; вместо них используются условные аббревиатуры.
Также из кода удалены точные названия переменных, например наименования целевых симптомов, лабораторных показателей и дозировок.
Для запуска кода необходимо наличие используемых доменов, а также замена скрытых переменных на соответствующие в вашем SDTM.
